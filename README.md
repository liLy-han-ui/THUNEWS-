# THUNEWS中文文本分类
1. 数据集：清华NLP组提供的THUCNews新闻文本分类数据集的一个子集，共30000条数据，涵盖教育，体育，财经，科技，家具，房产六个维度。每个类别5000条数据。
2. 在这次操作中，主要使用了三种方法进行中文文本分类：朴素贝叶斯算法，KNN算法，梯度提升决策树（Gradient Boosting Decision Tree, GBDT）模型。对于每一个模型，使用了均衡划分数据集和非均衡划分数据集两组数据集进行实验。
3. 在模型评估时，对验证集和测试集都进行评估，通过评估验证集上的表现，可以了解模型是否过拟合或欠拟合训练数据，并据此做出相应的调整；同时可以避免过拟合：如果仅依赖于训练集上的性能来进行模型调整，可能会导致模型过拟合训练数据，即模型在训练集上表现很好，但在未见过的数据（如测试集）上表现不佳。因此，使用验证集可以帮助识别并缓解这种过拟合现象。对验证集和测试集都进行评估，有助于全面了解模型的表现，确保所选模型既能在已知数据上有效工作，又具备应对新数据的能力。这整个过程是机器学习项目中确保模型质量和可靠性的重要环节。
4. 模型评估时输出了模型的分类报告，宏平均 F1 分数 (macro_f1) 和 微平均 F1 分数 (micro_f1)，以及混淆矩阵。
